# 数字图像处理的概念
- 图像是一个二维亮度函数f(x,y),(x,y)定义了空间坐标，f(x,y)定义该点的亮度或灰度
- 数字图像是指图像f(x,y)在空间坐标和亮度的数字化。数字图像是由有限的元素组成的，每一个元素都有一个特定的位置和幅值,这些元素称为图像元素或像素。
- 数字图像处理是指借用数字计算机处理数字图像。

# 图像处理的起源
- 数字图像处理最早的应用之一是在报纸业
- 1929年，能够编码15个灰度级的图像
- 现代数字图像处理的历史与数字计算机的发展密切相关。要求非常大的存储和计算能力

# 数字图像处理的应用
- 天文：星云图
- 医学：X光图像；大脑CT图像；
- 工业生产：集成电路；胶囊；
- 遥感、遥测
- 安全、身份识别
